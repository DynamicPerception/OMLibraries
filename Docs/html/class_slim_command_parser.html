slimCommand parse ( QString &  p_String)

Parse Command From String

Parses a scripting command and sends it to the indicated device if such a command is supported by the indicated device.

If the command begins with an alias for a device, it is issued to that device - otherwise, the default device port and address is used.

Exceptions:
SLIM_ERR_EMPTYNo command was given
SLIM_ERR_NODEVNo such device is registered
SLIM_ERR_NOCMDNo such command associated with device
Returns:
A slimCommand structure representing the command issued.
